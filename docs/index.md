# JSTQB Foundation Level 学習コンテンツ

JSTQB Foundation Level（テスト技術者資格 Foundation Level）の合格を目指す、非公式の学習コンテンツです。シラバス Version 2023 V4.0.J02 に沿った全 6 章 30 レッスンと、本番形式のドリルで構成しています。

::: warning 非公式・非公認の教材です
本教材は JSTQB Foundation Level シラバスを参考に作成した非公式の学習コンテンツであり、ISTQB®／JSTQB® が承認・公認したものではありません。正確な内容は必ず公式シラバスをご確認ください。詳しくは [出典・免責について](/about/) を参照してください。
:::

## JSTQB Foundation Level とは

- JSTQB（Japan Software Testing Qualifications Board）が実施する、ソフトウェアテスト技術者向けの資格試験です
- 出題範囲は公式シラバス（Foundation Level シラバス V4.0）に基づきます。このサイトの章立てはシラバスの 6 章に対応しています
- 試験は選択式 40 問・60 分・合格ライン 65%（40 問中 26 問）です
- 各学習目標には K1（記憶）・K2（理解）・K3（適用）の認知レベルが設定されています
- 最新の試験概要・シラバスは [JSTQB 公式サイト](https://www.jstqb.jp/syllabus/) で確認してください
